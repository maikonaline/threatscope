ThreatScope v1.0

Sistema de detección de amenazas en tiempo real desarrollado como proyecto de aprendizaje práctico orientado a operaciones SOC. Combina ML, threat intelligence y automatización de alertas.

IMPLEMENTADO
✓Pipeline completo: ingesta CSV → ML → threat intel → alerta
✓Detección de anomalías: Isolation Forest (scikit-learn)
✓Threat Intelligence: AbuseIPDB API + validación anti-SSRF
✓Mapeo MITRE ATT&CK: 17 técnicas y subtécnicas
✓Alertas automáticas: Slack webhook (activo en producción)
✓Código de notificación para Splunk HEC y Microsoft Teams
✓API REST: FastAPI + 48 tests automatizados (unit/integration/e2e)
✓Deploy: Docker + Render (backend) + Vercel (frontend)

ℹ︎ ¿Para qué sirve este análisis?
Carga un CSV con registros de tráfico de red y el motor de ML (Isolation Forest) detectará automáticamente comportamientos anómalos: ataques de fuerza bruta, escaneos de puertos, exfiltración de datos y más. Cada detección se enriquece con inteligencia de amenazas (AbuseIPDB) y se mapea a técnicas MITRE ATT&CK.
